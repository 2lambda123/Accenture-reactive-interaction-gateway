    request_passage(endpoint, ip \\ nil, opts \\ []) View Source 
    request_passage(String.t(), String.t() | nil, %{} | []) :: :ok | :passage_denied

    Request passage to a specific endpoint, from a given IP.

    Depending on the granularity, the endpoint might be the hostname of the target host, or the target socket, e.g., hostname <> ":" <> port.

    Calling this function always cause a side effect: internally, the request is recorded. Depending on previous calls, the function returns either :ok or :passage_denied. The latter means that the rate limit for the given endpoint (and source IP if per_ip? is true) has been reached, which means that the request in question should be blocked.
